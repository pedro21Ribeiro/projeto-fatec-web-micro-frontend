# Projeto de microfront-end

Na aula de programação web da Fatec de Mogi Mirim, primeiro semestre de 2025
foi proposto ao nosso grupo:

- Eder
- Mariane
- Moises
- Pedro Silva

Que construíssimos um projeto utilizando Svelte, devido a ser uma tecnologia que nós não
conhecemos. Porém, acabou sendo acrescentado ao projeto uma complexidade a mais como uma
forma de desafio. Devido a isso a nossa tech stack atual é composta por:

- Laravel (Papel de Backend para o Svelte)
- Svelte (Apresentando, alguns dados para o cliente)
- Java+HTMX (Fazendo o papel de autenticador e criação de itens)

O tema escolhido pelo nosso projeto foi: A criação de uma biblioteca SIMPLES
